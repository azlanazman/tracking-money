How to set up Google Sheets export (one time only)

  1. Open Google Sheets and create a new spreadsheet
  2. Click Extensions → Apps Script
  3. Delete everything and paste the code from sheets-script.gs
  4. Click Deploy → New deployment → Web app
  5. Set Execute as: Me · Who has access: Anyone
  6. Click Deploy, copy the Web App URL, and paste it above
